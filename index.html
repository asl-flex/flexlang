FLEX

The multi-purpose programming language for everything from scripting to systems programming

Getting Started

Run your first FLEX program in seconds:

// hello.flex
println("Hello, World!");

// Variables and functions
x = 10;
fnx double(n) {
    return n * 2;
}

println(double(x));

Compile and Run

# Compile to AST
./flex.sh hello.flex

# Run the program
./flex.sh -s hello.flexAT
